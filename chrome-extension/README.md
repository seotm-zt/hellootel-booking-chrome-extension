# TopTravel Booking Saver

Chrome-расширение для страницы [`https://toptravel.tours/order.html`](https://toptravel.tours/order.html), которое:

- добавляет красную кнопку `Добавить бронь в базу` на каждую карточку брони;
- парсит основные поля из карточки;
- сохраняет брони в локальную базу расширения через `chrome.storage.local`;
- показывает сохраненные записи в popup;
- работает только при активной авторизации на `toptravel.tours`;
- умеет экспортировать базу в JSON.

## Что парсится

- код брони;
- название отеля;
- подзаголовок карточки;
- даты проживания;
- гости;
- питание;
- трансфер;
- цена;
- статусы;
- ссылка на детали;
- URL страницы и время сохранения.

## Установка

1. Откройте `chrome://extensions`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked`.
4. Выберите папку проекта `D:\Alexcg.oline\chrome`.

## Как использовать

1. Откройте страницу `https://toptravel.tours/order.html`.
2. На каждой карточке брони появится красная кнопка.
3. Нажмите кнопку, чтобы сохранить бронь в локальную базу расширения.
4. Откройте popup расширения, чтобы посмотреть, удалить или экспортировать сохраненные брони.

## Структура

- `manifest.json` - конфигурация MV3-расширения;
- `icons/` - иконка расширения в размерах `16/32/48/128` и исходник `icon-master.png`;
- `content.js` - инъекция кнопки и парсинг страницы;
- `content.css` - стили красной кнопки и уведомлений;
- `popup.html`, `popup.css`, `popup.js` - интерфейс локальной базы броней.
